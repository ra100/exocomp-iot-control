# Exocomp remote control

> control "voice" lights, phaser and random blinking leds

| Pin | Function          |
| --- | ----------------- |
| D1  | Voice pulse       |
| D2  | Phaser fire       |
| D3  | Random blink      |
| D4  | onboard "WiFi on" |
| D5  | Random blink      |
| D6  | Random blink      |
| D7  | Random blink      |
| D8  | Random blink      |
